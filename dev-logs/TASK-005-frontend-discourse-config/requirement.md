# TASK-005: 前端 Discourse 配置界面

## 需求背景

用户需要在前端界面配置 Discourse 论坛的连接信息，包括全局设置和创建 Agent 时的个性化配置。

## 具体需求

### 1. 设置页面增加 Discourse 区域
- 论坛地址输入框
- API Key 输入框（密码类型，脱敏展示）
- 默认发帖分类选择

### 2. 创建 Agent 页面增加论坛配置
- 可选：为该 Agent 指定独立的论坛用户名
- 显示将使用的论坛地址（继承自全局设置）

## 验收标准
- [ ] 设置页面可配置 Discourse 信息
- [ ] 创建页面可指定 Agent 论坛用户名
- [ ] API Key 脱敏展示
