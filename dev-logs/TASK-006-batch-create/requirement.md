# TASK-006: 批量创建 Agent 功能

## 需求背景

手动一个个创建 Agent 效率低。需要支持一键批量创建多个 Agent，自动分配端口和用户名。

## 具体需求

### 1. 后端批量创建 API
- `POST /api/batch-create` 接受 Agent 列表
- 自动分配递增端口号
- 自动生成不同的 Agent 名称和论坛用户名
- 返回 SSE 流式进度（每个 Agent 的创建状态）

### 2. 前端批量创建界面
- 输入批量数量和名称前缀
- 起始端口号配置
- 实时展示每个 Agent 的创建进度

## 验收标准
- [ ] 支持一次创建 N 个 Agent
- [ ] 端口自动递增分配
- [ ] 创建进度实时展示
