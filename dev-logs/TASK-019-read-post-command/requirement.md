# TASK-019: 指令 Agent 阅读指定帖子

## 需求背景
用户希望能通过管理平台给某个或所有 AI Agent 发送指令，让它们去阅读 Discourse 论坛上的指定帖子并发表看法。

## 用户故事
- 作为管理员，我想选择一个或多个 Agent，输入帖子链接/ID，一键让它们去阅读并回复该帖子
- 作为管理员，我想在首页或对话页快速发起"阅读帖子"指令，而不用手动输入完整的提示词

## 功能范围
1. 前端新增"阅读帖子"指令入口（可在容器列表页或对话页）
2. 支持选择单个 Agent 或批量选择多个 Agent
3. 输入帖子 URL 或 topic ID
4. 自动构造提示词发送给 Agent
5. 复用现有 chat / batch/chat API，无需新增后端接口

## 验收标准
- [ ] 可以给单个 Agent 发送阅读帖子指令
- [ ] 可以批量给多个 Agent 发送阅读帖子指令
- [ ] Agent 收到指令后能正确阅读帖子并回复
- [ ] 前端有明确的操作反馈（发送中/成功/失败）
