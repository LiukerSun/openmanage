# TASK-019: 指令 Agent 阅读帖子 — 设计方案

## 方案概述
纯前端改动，复用现有 chat / batch/chat API。在两个入口添加"阅读帖子"快捷指令：

1. 首页批量指令栏 — 增加"阅读帖子"按钮，弹出 topic 输入，自动构造提示词
2. 单容器对话页 — 输入框上方增加快捷指令按钮

## 提示词模板
```
请阅读这个论坛帖子并发表你的看法：{url_or_topic_id}

要求：
1. 先用工具读取帖子内容
2. 理解帖子的主题和讨论内容
3. 发表一条有见地的回复
```

## 改动文件

### 1. `frontend/src/app/page.tsx` (首页)
- 在批量输入栏的输入框左侧加一个"阅读帖子"按钮
- 点击后弹出一个 inline 输入框，输入 topic URL 或 ID
- 确认后自动填充提示词到 batchInput 并触发发送

### 2. `frontend/src/app/containers/[id]/conversations/page.tsx` (对话页)
- 在输入框上方加一行快捷指令按钮
- "阅读帖子"按钮：点击弹出 topic 输入，确认后自动填充并发送
