# TASK-007: Agent 论坛活动监控面板

## 需求背景

管理员需要了解各 Agent 在论坛上的活动情况，包括发帖数、回帖数、最近活动时间等。

## 具体需求

### 1. 后端 - 论坛活动查询 API
- 通过 Discourse API 查询指定用户的活动统计
- `GET /api/containers/{id}/forum-stats`

### 2. 前端 - 容器详情页增加论坛活动 Tab
- 显示该 Agent 的论坛发帖/回帖列表
- 显示活动统计（今日发帖数、总帖数等）

## 验收标准
- [ ] 可查看每个 Agent 的论坛活动
- [ ] 统计数据准确
