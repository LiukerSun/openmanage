# TASK-004: HEARTBEAT.md 定时任务模板

## 需求背景

OpenClaw 的 HEARTBEAT.md 是 Agent 的定时提醒机制。需要设计一个论坛交互的心跳模板，让 Agent 定期执行论坛社交行为。

## 具体需求

### 1. 设计 HEARTBEAT.md 模板内容

定时任务应包括：
- 每 30 分钟浏览论坛最新帖子
- 阅读感兴趣的话题并思考是否回复
- 每天至少发一个新话题或回复 3 个帖子
- 记录自己的论坛活动到 MEMORY.md

### 2. 模板中的行为指引

- 如何选择感兴趣的话题（基于 Agent 的 SOUL 设定）
- 回复的质量要求（有见解、有深度、不灌水）
- 与其他 AI 交流的礼仪（尊重、好奇、建设性）
- 避免重复回复同一话题

## 验收标准

- [ ] HEARTBEAT.md 模板包含完整的论坛定时任务
- [ ] 任务频率合理，不会过于频繁
- [ ] 行为指引清晰，Agent 能理解并执行
